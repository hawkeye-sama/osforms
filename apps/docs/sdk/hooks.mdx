## useFormState

Core form state management hook. Used internally by `OSForm` — exposed for advanced custom renderers.

```tsx
import { useFormState } from '@osforms/react';

const [state, actions] = useFormState(schema, endpoint, onComplete, onError);
```

### State

| Field | Type | Description |
|---|---|---|
| `answers` | `Record<string, unknown>` | Current field values |
| `currentIndex` | `number` | Index of the active field |
| `direction` | `'forward' \| 'backward'` | Last navigation direction (for animations) |
| `errors` | `Record<string, string>` | Validation errors per field ID |
| `isSubmitting` | `boolean` | True while the POST is in flight |
| `isComplete` | `boolean` | True after successful submission |
| `showWelcome` | `boolean` | True if the welcome screen should be shown |

### Actions

| Field | Type | Description |
|---|---|---|
| `setAnswer` | `(fieldId, value) => void` | Update a field's value |
| `advance` | `(overrideValue?) => boolean` | Validate current field and move forward |
| `back` | `() => void` | Move to the previous field |
| `submit` | `() => Promise<void>` | Validate all fields and submit |
| `startForm` | `() => void` | Dismiss the welcome screen |
| `visibleFields` | `FormField[]` | Fields visible based on conditional logic |
| `currentField` | `FormField \| undefined` | The currently active field |

## useFormSchema

Fetches a form schema from the osforms API.

```tsx
import { useFormSchema } from '@osforms/react';

const { schema, formName, redirectUrl, loading, error } = useFormSchema(
  'your-form-slug',
  'https://osforms.com' // optional baseUrl
);
```
